## Usage
Create an element and add the class `btn` and `fab` to it. Inside that add the icon.
<div class="p-4 m-1 bg-dark-1">
	<button class="fab ripple-e">
		<i class="mdi mdi-heart"></i>
	</button>	
</div>

```xml
<button class="fab ripple-e">
	<i class="mdi mdi-heart"></i>
</button>	
```
